# **App Name**: UniLife Organizer

## Core Features:

- Interactive Calendar: A visual calendar interface to view daily, weekly, and monthly schedules.
- Task Management: Create, assign categories, and manage tasks with due dates, priorities, and completion status.
- AI-Powered Scheduling Tool: Suggest optimal times to schedule tasks based on class schedule, work schedule and free time. It should reason whether and when to remind the user about specific appointments.
- Notifications & Reminders: Set up notifications for upcoming tasks, classes, and work shifts to stay on schedule.
- Category Creation: Create named categories like Work, Classes, Extracurricular, Social, and Personal to associate with your schedule events and filter calendar view.
- Notes: Create and manage notes.

## Style Guidelines:

- Primary color: Soft green (#C8E6C9) to promote a calm and organized feeling.
- Background color: Very light green (#F1F8E9), almost white, for a clean and fresh look.
- Accent color: Olive green (#A5D6A7) for highlights and key actions, creating a subtle contrast.
- Dark Mode: Dark grey background with light green accents for comfortable nighttime use.
- Body and headline font: 'Alegreya', a serif font providing an elegant, intellectual, contemporary feel.
- Use minimalist, line-based icons in olive green to represent different categories of tasks and events.
- Clean, modern layout with ample whitespace to avoid clutter and ensure ease of use.
- Subtle transitions and animations for a smooth and engaging user experience.
